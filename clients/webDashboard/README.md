# Municipality Ticketing Web Dashboard

Belediye Otobüs Bilet Sistemi - Yönetim Paneli

## Teknoloji Stack

- **React 19** - UI Framework
- **TypeScript 5.6+** - Type Safety
- **Vite 6** - Build Tool
- **Zustand** - Client State Management
- **TanStack Query 5** - Server State & Caching
- **React Router 7** - Client-side Routing
- **shadcn/ui** - Component Library (Planlanıyor)
- **Tailwind CSS 4** - Styling (CSS-based configuration)
- **Axios** - HTTP Client
- **Recharts 2.15+** - Charts & Analytics (v3'e migration recommended)
- **SignalR** - Real-time Updates
- **Lucide React** - Icons
- **date-fns** - Date utilities

## Kurulum

### Gereksinimler

- Node.js 20+ 
- npm veya yarn
